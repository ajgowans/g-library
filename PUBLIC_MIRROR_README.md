# Gotham Library Public Mirror

This folder is generated from the private Gotham Library working repo.
It is intended for sharing public-safe reports and reference CSVs with GPT Classic or a public GitHub mirror.

The private working repo remains the source of truth.

## Publication Status

- No licence file is included yet. Add a licence before publishing this as a public repo.
- Raw spreadsheet exports, order ledgers, copy ledgers, check-in commands, and generated phone lookup files are excluded by default.
- Review `PUBLIC_MIRROR_MANIFEST.csv` before publishing.
